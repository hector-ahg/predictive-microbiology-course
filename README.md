# Predictive Microbiology Workshop: from microbiomes to models for research (UNAM)

This repository contains materials for a workshop on predictive microbiology using Python and machine learning.

## Structure

- 'notebooks/' – course content
- 'pipeline/' – QIIME2 and PICRUSt2 workflows (reference only)
- 'scripts/' – helper functions
- 'data_links.md' – dataset download (Zenodo)

## Dataset

Data is hosted externally (Zenodo) to keep the repository lightweight.

## Usage

Open notebooks in Google Colab (recommended) or run locally.
